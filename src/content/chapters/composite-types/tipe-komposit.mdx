Sebelumnya kita sudah berkenalan dengan tipe data seperti integers, 
string, float, Boolean. Tipe data tersebut merupakan tipe data umum yang 
berada pada sebuah pemrograman tipe data ini juga disebut tipe data 
primitif. Dalam pemrograman, tipe komposit adalah tipe data yang terdiri 
dari tipe data umum. Tipe komposit ini dapat digunakan untuk 
mengelompokkan data terkait, sehingga lebih mudah untuk dikelola dan di 
manipulasi dalam kode Anda.
Golang sendiri memiliki beberapa tipe data komposit yang memungkinkan 
pengembang untuk mengelola dan merepresentasikan data dengan 
berbagai tingkat kompleksitas.

Tipe data komposit ini memberikan dasar yang kuat untuk mengelola data 
dalam Golang, dengan masing-masing memiliki karakteristik dan kegunaan 
yang unik. Penggunaan tipe data komposit ini memungkinkan Anda untuk 
mengorganisasi dan merepresentasikan data dalam cara yang sesuai untuk 
masalah yang sedang Anda selesaikan dalam pemrograman Golang.
Selanjutnya Anda akan mempelajari tipe data komposit yang tersedia pada 
pemrograman Golang.
